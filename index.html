A Pseudo class in CSS is used to define the special state of an element. It can be combined with a CSS selector to add an effect to existing elements based on their states. For Example, changing the style of an element when the user hovers over it, or when a link is visited. All of these can be done using Pseudo Classes in CSS. Note that pseudo-class names are not case-sensitive. Syntax: selector: pseudo-class{ property: value; }

parent inherit

Parent inheritance is a feature of CSS that allows the properties of a parent element to be inherited by its child elements. This means that if you set a property on a parent element, the child elements will inherit that property unless you explicitly override it. For example, if you set the font-size property on a parent element to 16px, all of the child elements will also have a font size of 16px unless you explicitly set a different font size for the child elements.
